Education Platform API

Education Platform API

API для керування освітньою платформою. Дозволяє виконувати операції з курсами та студентами.

Основні можливості

  • Управління курсами (створення, редагування, видалення)
  • Управління студентами (реєстрація, оновлення профілю)
  • Запис студентів на курси
  • Перегляд списків студентів на курсах

Автентифікація

API використовує JWT токени для автентифікації. Отримайте токен через /auth/login (не входить в поточну специфікацію) та використовуйте його в заголовку Authorization: Bearer {token}.
