## Про проект

Цей проект створено в рамках вивчення специфікації OpenAPI та інструментів для документування REST API. Метою роботи було:

- Описати REST API для освітньої платформи
- Створити OpenAPI специфікацію в форматі YAML
- Згенерувати клієнтський SDK
- Налаштувати документацію за допомогою Zudoku
- Розгорнути документацію на GitHub Pages
