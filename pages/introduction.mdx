# Вступ до Education Platform API

Ласкаво просимо до документації API освітньої платформи. Цей API дозволяє керувати курсами та студентами, а також записувати студентів на курси.

## Про API

API побудовано на принципах REST та використовує стандартні HTTP методи:
- `GET` - отримання даних
- `POST` - створення нових ресурсів
- `PUT` - оновлення існуючих ресурсів
- `DELETE` - видалення ресурсів

## Основні сутності

- **Course** - навчальний курс з інформацією про викладача, кредити та опис
- **Student** - студент з персональними даними та списком курсів

## Формати даних

Всі запити та відповіді використовують формат JSON. Дати передаються у форматі ISO 8601.
